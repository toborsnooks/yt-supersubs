YouTube "Supersubs" Samples

This is a demo gallery of embedding YouTube videos, and augmenting them with additional subtitles that may or may not be available on YouTube proper. YouTube shut down community contributed translations in 2020, so this is a workaround. The following links all display the same video, but with subtitles in different languages.

The pipeline here is OpenAI Whisper -> .srt file -> Hugging Face MarianMT -> translated .srt
